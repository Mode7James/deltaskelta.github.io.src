## The Problem States

The <M i="n^th" /> term of the sequence of triangle numbers is given by, <M i="t_n =
\frac{1}{2}n(n + 1)" /> so the first ten triangle numbers are:

<M b="1, 3, 6, 10, 15, 21, 28, 36, 45, 55, ..." />

By converting each letter in a word to a number corresponding to its alphabetical position and adding these values we
form a word value. For example, the word value for SKY is <M i="19 + 11 + 25 = 55 = t_10" />. If the word
value is a triangle number then we shall call the word a triangle word.

Using [words](https://projecteuler.net/project/resources/p042_words.txt) (right click and 'Save Link/Target As...'),
a 16K text file containing nearly two-thousand common English words, how many are triangle words?

## My Solution

```python
# Opening the file and getting it displayed correctly
open_words = open('Euler42_words.txt')
words = open_words.read()
open_words.close()
words = words.split('&quot;')
#opening and splitting the words in the file

for i in words:
    if i == ',':
        words.remove(i)

# Defining a function to make a list of triangle numbers
def triangle(iters):
    triangles = []
    for i in range(1, iters + 1):
        tri = i * (i + 1) / 2
        triangles.append(tri)
    return triangles

triangles = triangle(30)
alpha = 'ABCDEFGHIJKLMNOPQRSTUVWXYZ'
# Defining triangles list and the alphabet

words_sum = []
for i in words:
    # Adding up the sums for every word
    alphasum = 0
    for j in i:
        alphasum += alpha.index(j) + 1
    words_sum.append(alphasum)        

count = 0
for i in words_sum:
    # Checking each sum to see if it is in the list of triangle numbers
    if i in triangles:
        count += 1

print count
```
